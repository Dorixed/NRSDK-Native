Detailed Description

Functions for swapchain.

Create the swap chain object and the associated native buffers.

Call NRSwapchainDestroy() to free the associated native buffers and the swap chain if no longer needed.

Parameters
[in]rendering_handleThe handle of rendering object.
[in]spec_handleThe handle of buffer specification object.
[out]out_swapchain_handleOpaque handle of the newly created swap chian object.
Returns
The result of operation.
