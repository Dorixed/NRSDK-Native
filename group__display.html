Detailed Description

Functions for display system.

Note: NRDisplayCreate function should be called to render the application content to glasses display, or it will be rendered to phone display.

Example:

NRHandle display_handle = NRHandleNull;
NRResult result = NRDisplayCreate(&display_handle);
// Handle result.
result = NRDisplayStart(display_handle);
// Handle result.
// when need to stop
result = NRDisplayStop(display_handle);
// Handle result.
result = NRDisplayDestroy(display_handle);
// Handle result.
// when pause
result = NRDisplayPause(display_handle);
// Handle result.
// when resume
result = NRDisplayResume(display_handle);
// Handle result.
